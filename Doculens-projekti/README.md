# DocuLens – TiCorporate-projekti (JAMK, syksy 2025)

## Projektin kuvaus
DocuLens oli JAMKin TiCorporate‑kokonaisuuden projektityö syksyllä 2025. Projektin tavoitteena oli kehittää selainpohjainen työkalu, joka helpottaa organisaatioiden dokumenttien hallintaa, analysointia ja löydettävyyttä. Ratkaisu suunniteltiin erityisesti tilanteisiin, joissa dokumenttimäärät kasvavat ja niiden rakenteellinen analysointi on hidasta tai manuaalista.

## Keskeiset ominaisuudet
- Dokumenttien automaattinen analysointi  
  Rakenteen, metatietojen ja sisällön tunnistaminen.
- Hakutoiminnot ja suodattimet  
  Dokumenttien nopea selaus ja luokittelu.
- Selkeä käyttöliittymä  
  Suunniteltu helppokäyttöiseksi myös ei‑teknisille käyttäjille.
- Modulaarinen arkkitehtuuri  
  Mahdollistaa ominaisuuksien laajentamisen ja integroinnin muihin järjestelmiin.

## Teknologiat
- Angular 
- TypeScript  
- JavaScript
- Node.js / Express  
- GitHub Projects
- MySQL database
- AWS services

## Roolini projektissa
Toimin projektissa backend‑kehittäjänä ja Product Ownerina. Vastuualueisiini kuuluivat:

- Teknisen arkkitehtuurin suunnittelu  
- Dokumenttien käsittelylogiikan toteutus  
- Projektin backlogin luominen ja ylläpitäminen

## Projektin tavoitteet
- Rakentaa toimiva prototyyppi, joka demonstroi dokumenttien analysoinnin automatisointia.  
- Luoda modulaarinen ja laajennettava koodiarkkitehtuuri.  
- Tuottaa selkeä dokumentaatio ja esittelymateriaali TiCorporate‑demotilaisuuteen.

## Tulokset
- Valmis prototyyppi, joka mahdollisti dokumenttien lataamisen, analysoinnin ja selaamisen.   
- Selkeä UI‑rakenne ja modulaarinen projektipohja jatkokehitystä varten.  
- Projektista saatiin positiivista palautetta TiCorporate‑ohjaajilta.
